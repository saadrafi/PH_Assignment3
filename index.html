1.Difference between Flexbox and Grid?

Flexbox is made for one-dimensional layouts, like a row or a column. Grid is made for two-dimensional layouts of any size or dimension. Flexbox is a great solution for simpler one-dimensional layouts, like most navigations, sliders, and forms. Grid is a great solution for complex two-dimensional layouts, like those found in magazine layouts, image galleries, and dashboards.

2.What is the purpose of media query?

Media queries are a key part of responsive web design. They allow you to target CSS rules based on - for instance - screen size, device-orientation or display-density. This means you can use CSS Media Queries to tweak a CSS for an iPad, printer or create a responsive site.For example, you can apply styles specific to large displays if the user is accessing the page on a desktop. You can then apply other styles specific to small displays if the user is accessing the page on a mobile device.

3.What is CSS box model and how it works?

The CSS box model is essentially a box that wraps around every HTML element. It consists of: margins, borders, padding, and the actual content. The box model is essentially a box that wraps around every HTML element. It consists of: margins, borders, padding, and the actual content.

  • The padding and border are drawn outside of the content area, and increase the total width and height of the box.
  • The margin is drawn outside of the border, and increases the total width and height of the box.
4.What is semantic tag?

Semantic HTML is the use of HTML markup to reinforce the meaning of the information in webpages. Rather than simply div and span elements dividers and placeholders, semantic HTML provides web developers with elements like article, aside, details, figcaption, figure, footer, header, main, mark, nav, section, summary, and time. These elements provide meaning to the web page, and can be used by browsers, search engines, and other web developers. The use of these elements indicates the purpose of the information that they surround.
